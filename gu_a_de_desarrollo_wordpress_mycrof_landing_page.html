# Guía de Desarrollo: Landing Page MYCROF (WordPress Template) Esta guía detalla la estructura técnica, animaciones y micro-interacciones para desarrollar la nueva landing page de MYCROF como un template de WordPress. ## 1. Stack Tecnológico Recomendado - **WordPress** (Última versión) - **Editor:** Gutenberg con bloques personalizados o Elementor/Divi si se prefiere maquetador. - **Tipografía:** Poppins (vía Google Fonts). - **Iconografía:** Lucide React o FontAwesome (estilo lineal/tecnológico). - **Animaciones:** Framer Motion o GSAP para mayor control de scroll. ## 2. Paleta de Colores & Estilo - **Primary:** #627DE9 (Azul vibrante) - **Secondary:** #461CBC (Púrpura intenso) - **Dark:** #282E2F (Casi negro) - **Background:** #FFFFFF / #F8FAFC (Blanco/Gris muy claro) - **Gradient:** `linear-gradient(135deg, #627DE9 0%, #461CBC 100%)` ## 3. Estructura de Secciones (Conversión) ### Hero Section (El Gancho) - **Visual:** Fondo con malla degradada sutil o partículas tecnológicas en movimiento. - **Contenido:** - Título (H1 Poppins Black): "Conectando el Mejor Talento IT con el Futuro." - Slogan: "Uploading Talent" (con efecto de cursor de escritura). - CTAs Duales: [Contratar Talento] (Primario) [Soy Talento] (Secundario/Esquema). - **Animación:** Fade-in de abajo hacia arriba al cargar. ### Servicios & Valor (Qué hacemos) - **Layout:** Grid de 3 columnas. - **Micro-interacción:** Cards que se elevan y muestran un borde con el gradiente de marca al hacer hover. - **Contenido extraído:** Reclutamiento especializado, Capacitación IT, Consultoría de Talento. ### Proceso Tecnológico (Cómo lo hacemos) - **Visual:** Línea de tiempo vertical o stepper horizontal interactivo. - **Animación:** Los pasos se activan (iluminan) según el scroll del usuario. ### Captación de Talento (Para profesionales) - **Sección:** Fondo oscuro (#282E2F) con el logo en isotipo (símbolo) como marca de agua. - **CTA:** Botón con brillo pulsante: "Únete a la Red MYCROF". ### Formulario de Contacto (Conversión Final) - **Diseño:** Minimalista, campos con bordes finos que cambian a azul vibrante al enfocarse. - **Micro-interacción:** Mensaje de éxito animado (check-mark). ## 4. Instrucciones para Agentes de Desarrollo 1. "Crea un custom post type para 'Servicios' que permita gestionar las tarjetas de la landing." 2. "Implementa un shader de WebGL sutil para el fondo del Hero usando la paleta #627DE9 y #461CBC." 3. "Asegura que todos los botones tengan una transición de 0.3s con curva cubic-bezier." 4. "Optimiza las imágenes para carga rápida (WebP) y usa Lazy Loading."
